# AI Materials Discovery Platform - 상세 구현 설명 (최종본)

본 프로젝트는 제공된 실험 데이터(`stmech_aus_ss.xls`)를 포함하여 사용자가 원하는 어떤 신소재 실험 데이터라도 AI로 즉시 분석하고 물성을 예측할 수 있도록 설계된 통합 플랫폼입니다.

---

## 1. 주요 기능 개선 사항
- **동적 데이터 로딩**: 고정된 파일이 아닌, 사용자가 직접 탐색기에서 엑셀 파일을 선택하여 실시간 분석이 가능하게 개선되었습니다.
- **한글 폰트 지원**: 모델 학습 그래프 내 한글 깨짐 현상을 해결하여 정밀한 분석 시각화를 지원합니다.
- **예측 서버 연동 최적화**: 학습 즉시 서버가 새 모델을 인식하여 오차 없이 결과를 반환하도록 자동 리로드 기능을 탑재했습니다.

## 2. 프로젝트 폴더 구조
- `src/data_pipeline/parser.py`: 범용 엑셀 파서 (데이터 전처리 및 정규화)
- `src/models/ai_model.py`: TFP(TensorFlow Probability) 기반 베이지안 가우시안 신경망
- `src/api/app.py`: 추론용 기반 Flask 서버
- `src/gui/main_window.py`: 사용자 친화적 PyQt6 GUI 클라이언트 (한글화 완료)
- `main.py`: 서버와 GUI를 한 번에 구동하는 통합 실행 파일

---

## 3. 프로그램 실행 방법 (How to Run)

본 플랫폼은 가상환경(venv) 내의 모든 의존성이 설정되어 있습니다. 다음 단계를 따라 실행하세요.

1. **프로젝트 폴더로 이동** (터미널/CLI):
   ```bash
   cd c:\ai_project\ai-materials-discovery-platform
   ```

2. **통합 실행 파일 구동**:
   ```bash
   # 가상환경의 python을 사용하여 main.py를 실행합니다.
   .\venv\Scripts\python.exe main.py
   ```

3. **작동 확인**:
   - 실행 후 잠시 기다리면 **Flask API 서버**가 백그라운드에서 켜지고, 이어서 **GUI 어플리케이션 창**이 화면에 나타납니다.

---

## 4. 권장 사용 시나리오

1. **[데이터] 탭**: **[파일 선택]**을 눌러 분석할 엑셀 파일을 고른 후 **[데이터 구조 분석]**을 실행합니다.
2. **[모델 학습] 탭**: **[AI 모델 학습 시작]**을 클릭합니다. 그래프의 한글이 잘 나오는지 확인하며 오차가 줄어들 때까지 기다립니다.
3. **[물성 예측] 탭**: 18종의 합금 성분을 직접 입력하거나, **[📋 분석 데이터 샘플로 채우기]** 버튼을 눌러 실제 데이터 중 하나를 자동으로 채울 수 있습니다. 이후 **[예측]** 버튼을 눌러 AI의 항복강도 예측치와 성분 기여도 그래프를 분석합니다.
4. **[최적 소재 설계] 탭**: **[최적 조성 도출 시작]**을 눌러 AI가 제안하는 고강도 '황금 비율'을 확인합니다.
5. **[AI 결과 보고서] 탭 (신설)**: 모든 분석 결과에 대한 AI의 상세 설명과, 특히 **최적화 수치(533 MPa)가 도출된 논리적 과정**을 한글 보고서로 최종 확인합니다.

## 5. AI 학습이란 무엇인가? (개념 설명)

인공지능(AI) 플랫폼에서 '학습'을 한다는 것은 다음과 같은 원리입니다.

1. **기출문제 풀기**: AI에게 과거의 실험 데이터(성분과 온도에 따른 실제 강도 결과) 수천 건을 '기출문제'처럼 줍니다.
2. **패턴 파악**: "아, 니켈(Ni) 성분이 높아질수록 강도가 세지는 경향이 있구나!" 혹은 "온도가 너무 높으면 오히려 강도가 떨어지는구나!"와 같은 **숨겨진 수학적 규칙**을 스스로 찾아냅니다. 
3. **오차 줄이기 (그래프의 의미)**: 처음에는 AI도 찍기 때문에 틀립니다. 하지만 여러 번 반복학습(Epochs)하면서 정답과의 차이(Loss, 오차)를 점점 줄여나갑니다. 그래프가 아래로 내려가는 것은 AI가 점점 똑똑해지고 있다는 뜻입니다.
4. **실전 응용 (추론)**: 학습이 끝나면 AI는 이제 한 번도 본 적 없는 성분 조합을 보더라도, 자신이 배운 규칙을 바탕으로 결과를 매우 정확하게 맞힐(예측할) 수 있게 됩니다.

## 6. 항복강도(Yield Strength)란 무엇인가?

AI가 예측해주는 '항복강도(MPa)'는 소재의 **버티는 힘**을 나타내는 핵심 지표입니다.

1. **되돌아오는 한계점**: 금속을 잡아당기면 처음에는 고무줄처럼 늘어났다가 놓으면 원래대로 돌아갑니다. 하지만 어느 지점을 넘어서면 **영구적으로 변형**되어 돌아오지 않는데, 바로 그 '한계점'이 항복강도입니다.
2. **비유 (클립)**: 종이 클립을 살짝만 벌리면 다시 원래 모양으로 돌아오지만, 너무 크게 벌리면 휘어진 채로 남죠? 그 휘어지기 시작하는 순간의 힘이 바로 항복강도입니다.
3. **설계의 기준**: 자동차나 교량을 만들 때, 부품이 휘어버리면 큰일 나겠죠? 그래서 엔지니어들은 소재의 항복강도보다 훨씬 낮은 힘만 가해지도록 부품을 설계합니다.
4. **수치의 의미(MPa)**: 숫자가 클수록 더 큰 힘을 견딜 수 있는 '강한 소재'라는 뜻입니다.
